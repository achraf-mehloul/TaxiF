## المتطلبات
- Python 3.11 أو أحدث

## خطوات التشغيل

### 1. أنشئ بيئة افتراضية

#### على Linux / MacOS
```bash
python3.11 -m venv myenv
source myenv/bin/activate
```

#### على Windows (PowerShell)
```powershell
python -m venv myenv
.\myenv\Scripts\activate
```

### 2. ثبّت المكتبات
```bash
pip install -r requirements.txt
```

### 3. تشغيل البوت
```bash
python3.11 bot.py
```
